# Elite Gaming – Stability Fix Pack (2026‑03‑03)

This build focuses on **making the app reliably work** across Supabase schema changes (the #1 cause of "white screen" and missing data).

## What was fixed

### 1) Profiles table mismatch
Some pages queried `public.profiles` while the admin console schema uses `public.eg_profiles`.

**Fix:** the app now uses `src/lib/profileRepo.ts` which:
- tries `eg_profiles` first
- falls back to `profiles`
- supports the old shape where `profiles.id = auth.users.id`

### 2) Fixtures list reliability
The fixtures list previously relied on brittle PostgREST relationship embeds.

**Fix:** `useFixtures` now prefers stable views:
- `eg_fixtures_with_teams` (recommended)
- fallback: `eg_fixtures_by_stage`
- fallback: `eg_fixtures`

### 3) Stat leaders cache is now competition-aware
Switching competitions could show old leaders because cached keys didn’t include the season.

**Fix:** cache keys are now namespaced by `competitionKey:seasonSlug`.
